[БАЗОВАЯ ИНСТРУКЦИЯ]
1) Установить драйвер CH341SER.7z из папки drivers
2) Подключить устройство по USB к ПК
3) Используя ярлык "Edit Custom Data" записать необходимую пользователю информацию. При необходимости добавить картинки в файл - использовать ярлык "Convert Image"
4) Использовать ярдык "Start Toolchain Lite" для загрузки пользовательской информации
5) Отключить устройство от ПК


*****************************************************
		ПОДРОБНАЯ ИНФОРМАЦИЯ		
*****************************************************

[ДРАЙВЕРЫ]
CH341SER.7z - драйвер необходимый для подключения устройства к ПК.

[ЯРЛЫКИ]
"Edit Custom Data" - открыть файл для редактирования пользовательских данных
"Reset Device" - перезапустить устройство
"Convert Image" - запустить утилиту для конвертирования изображений
"Start Toolchain Full" - запустить полную последовательность утилит для прошивки EEPROM устройства. производится перекомпиляция всех системных файлов.
"Start Toolchain Lite" - запустить минимальную последовательность утилит для прошивки в EEPROM устройства пользовательских данных. для ускорения используются ранее скомпилированные системные файлы. 

[ФАЙЛЫ]
[!] все файлы кроме "custom_data.txt" являются системными и их редактирование не требуется.

custom_data.txt - файл с пользовательской информацией, к которой можно получить доступ на устройсте. поддерживает текст в кодировке windows-1251 и предварительно конвертированные монохромные изображения в формате 128x32. максимальный файла объем составляет около 60КБ. для добавления изображения необходимо использовать image_converter.exe и преобразовать файл в последовательность цифр (код), которую затем можно вставить в любое нужное место в тексте. изображение обязательно должно быть изолировано с 2 сторон знаками "¶" в тексте, чтобы ПО устройства смогло корректно определить его начало и конец. таким образом знаки должны стоять в конце и в начале вставленного кода изображения, без пробелов и любых других знаков. Символ табуляции используется для отображения текста, следующего после данного символа, на новой странице.

core_font.bmp - системный шрифт в кодировке windows-1251, используемый устройством для отображения всей текстовой информации. 

core_logo.bmp -  системное изображение, которое отображается при запуске устройства в качестве заставки. 

core_strings.ini - хранит все системные надписи пользовательского интерфейса, которые использует ПО устройства. 


[УТИЛИТЫ]
image_converter.exe - конвертер изображений. позволяет конвертировать файлы формата BMP в монохромные изображения, пригодные для отображения на OLED-дисплее. 
Поддерживает 3 типа файлов: 
- core_font.bmp;
- core_logo.bmp; 
- любые файлы BMP-файлы в разрешении 128x32, которые могут быть вставлены в качестве картинок в тексте после перекодировки.

eeprom_builder.exe - из отдельных файлов собирает EEPROM-образ, готовый для прошивки в flash-память устройства. компилирует файлы core_strings.ini, core_logo.bin, core_font.bin и custom_data.txt вместе. 

eeprom_loader.exe - загружает созданный образ EEPROM-файла (eeprom.bin) в память устройства. Емеет дополнительную функцию для перезагрузки устройства.
